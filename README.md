# 悬壶承光 - 八字紫微命理小程序

## 项目简介
专业的八字、紫微斗数、西方占星命理分析小程序

## 功能特性
- ✅ 八字排盘（四柱、五行、十神、神煞）
- ✅ 紫微斗数（十二宫位、主星、四化）
- ✅ 西方占星（太阳星座、行星相位）
- ✅ AI深度解读（火山方舟 API）
- ✅ 积分系统
- ✅ 微信支付

## 技术栈
- 微信小程序原生开发
- 微信云开发（云函数 + 云数据库）
- lunar-javascript（八字计算）
- iztro（紫微斗数）
- astronomy-engine（西方占星）

## 部署步骤

### 1. 准备工作
- 注册微信小程序账号
- 开通微信云开发
- 获取云开发环境 ID

### 2. 导入项目
1. 下载微信开发者工具
2. 选择「导入项目」
3. 选择本项目目录
4. 填入你的 AppID

### 3. 配置环境
1. 修改 `app.js` 中的 `envId` 为你的云开发环境 ID
2. 部署云函数：
   - 右键 `cloud/functions/fateEngine`
   - 选择「创建并部署：云端安装依赖」

### 4. 上传发布
1. 点击「上传」按钮
2. 填写版本号和项目备注
3. 登录微信公众平台提交审核
